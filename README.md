# ad-rpi/homebrew-tap

Homebrew tap for [BrewBrowser](https://github.com/ad-rpi/BrewBrowser) — a native
macOS GUI for Homebrew.

## Install

```sh
brew tap ad-rpi/tap
brew install --cask brewbrowser
```

> BrewBrowser isn't notarized yet. On first launch macOS may block it —
> right-click the app and choose **Open**, or install with
> `brew install --cask --no-quarantine brewbrowser`.
